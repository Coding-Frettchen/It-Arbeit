Installer Images

Kali ist eine sogenannte Rolling-Distribution, was bedeutet, dass wir Updates sofort veröffentlichen, sobald sie verfügbar sind. Benutzer können aus einer Vielzahl von Abbildern wählen. Weitere Informationen finden Sie unter „Welches Abbild sollte ich herunterladen?“ und „Kali-Zweige“. Für die meisten Nutzer empfehlen wir das neueste „Point Release“-Abbild unten, außer in Fällen, in denen ein Benutzer eine spezifische Fehlerbehebung benötigt – in diesem Fall könnte der wöchentliche Build die beste Wahl sein. Wir erstellen vierteljährlich neue Kali Linux-Abbilder. Diese werden die offiziellen „Point Releases“. Diese Abbilder werden getestet und anschließend mit einem Blog-Beitrag angekündigt.

Per-built VMs

Kali Linux VMware- und VirtualBox-Abbilder sind für Benutzer verfügbar, die eine Installation in einer virtuellen Maschine bevorzugen oder spezifische Anforderungen haben. Diese Abbilder verwenden standardmäßig die Zugangsdaten „kali/kali“.

Mobile

Kali NetHunter Pro ist die offizielle Kali Linux-Version für mobile Geräte wie das Pine64 PinePhone und PinePhone Pro.

Cloud

Kali Cloud-Abbilder können schnell in verschiedenen Cloud-Infrastrukturen bereitgestellt werden. Dadurch ist ein einfacher Zugriff auf eine Kali-Installation in einer modernen Umgebung mit leistungsstarker Hardware möglich. Diese Abbilder werden häufig für kurzzeitige Kali-Installationen verwendet, die schnell eingerichtet und abgebaut werden können.

Containers

Mit Containern wie Docker und LXC/LXD können Kali-Container das Kali-Toolset auf Ihrem Host-Betriebssystem bereitstellen, ohne ein zusätzliches vollständiges Betriebssystem auszuführen. Es gibt jedoch Einschränkungen, z. B. keinen direkten Hardwarezugriff und komplizierte Handhabung eingehender Verbindungen.

Live

Ein Kali Linux Live-Abbild auf CD/DVD/USB/PXE ermöglicht den Zugriff auf eine vollständige Kali-Installation, ohne das bereits installierte Betriebssystem zu verändern. Dies bietet schnellen und einfachen Zugriff auf das Kali-Toolset mit allen Vorteilen einer Bare-Metal-Installation. Es gibt jedoch einige Nachteile, z. B. langsamere Speicheroperationen aufgrund des verwendeten Speichermediums. For most users, we recommend the latest “point release” image below, except in cases when a user requires a specific bug patch, in which case the weekly build may be best.

WSL

Das Windows Subsystem for Linux (WSL) ist ein Softwarepaket für moderne Windows-Installationen, das die parallele Ausführung von Linux unter Windows ermöglicht. Das Kali WSL-Paket bietet einfachen Zugriff auf das Kali-Toolset. Dies hat die gleichen Nachteile wie eine Standard-VM, jedoch mit weniger Overhead und besserer Integration.
